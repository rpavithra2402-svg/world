# World

A simple HTML website project hosted on GitHub Pages.

## Features
- Clean and responsive HTML structure
- Custom CSS styling with light blue theme
- JavaScript functionality with welcome message
- Perfect starting point for beginners

## Files
- **index.html** - Main HTML file with page structure
- **style.css** - CSS styling for the website
- **script.js** - JavaScript code for interactivity
- **README.md** - Project documentation

## Getting Started
1. Clone or download this repository
2. Open `index.html` in your web browser
3. You'll see the welcome alert and styled page

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
